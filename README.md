# منصة السيادة للذكاء الاصطناعي (Sovereign AI Platform v18.0.0)
### الاسم الرمزي: "المُعِزّ" (Al-Mu'izz) - Predator Edition

هذه هي المنصة الأكثر تقدماً للعمليات السيبرانية السيادية، مصممة لتحويل جهازك إلى منظومة "المُعِزّ - المفترس" المتكاملة التي تدمج الذكاء الاصطناعي مع العمليات الهجومية والدفاعية الغاشمة.

## 🚀 ما الجديد في نسخة المفترس (v18.0)؟
- **Alpha Node Orchestrator:** عقل مركزي يقوم ببناء سلاسل هجوم آمتتة (Autonomous Attack Chains).
- **AI Model Harvester:** محرك بحث واستحواذ على موديلات الذكاء الاصطناعي المحلية (Ollama) وتوظيفها فوراً.
- **GitHub Intel Sync:** دمج تقنيات FUD (غير قابلة للكشف) مستوحاة من أحدث مستودعات Red-Team العالمية.
- **Predator UI:** واجهة هجومية باللون الأحمر القتالي والأسود الفحمي مع تأثيرات النبض العصبي.

## 🚢 نظام "سفينة نوح" (Noah's Ark v3)
وحدة النسخ الاحتياطي والتعافي الشامل لضمان الصمود السيادي.
- **الموقع:** `scripts/sovereign_ark_v3.sh`
- **التشغيل:** من الواجهة (System > Noah's Ark) أو عبر التيرمينال: `sov-backup`.

## متطلبات التشغيل الأساسية
- **نظام التشغيل:** Kali Linux 2024+ (منصح به بشدة) أو Ubuntu 22.04+.
- **المعالجة:** يفضل وجود GPU لعمليات الاستدلال المحلي، مع 8GB RAM كحد أدنى.
- **المتغيرات البيئية:** يجب وضع `GEMINI_API_KEY` في ملف `.env` لتفعيل العقل المركزي.

## بروتوكول التثبيت السريع
1. **منح صلاحيات التنفيذ:**
   ```bash
   chmod +x install.sh
   chmod +x scripts/*.sh
   ```
2. **بدء عملية التكوين:**
   ```bash
   sudo ./install.sh
   ```

## كيفية التشغيل
- **واجهة العمليات (Web Dashboard):** `npm run dev` (توجه إلى: `http://localhost:9002`)
- **مركز القيادة (TUI):** `sovereign`
- **الاستحواذ على النماذج:** من صفحة "System Pulse" اضغط على "Harvest Local Assets".
