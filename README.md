5.  **إمبراطورية الظل (Shadow Grid):** إدارة الزومبيز.
6.  **وحدة الاشتباك (Mobile Strike):** السيطرة على الأجهزة المادية.
7.  **الاجتياح الاستخباري (Deep Recon):** تشريح البصمة الاجتماعية.
