# منصة السيادة للذكاء الاصطناعي (Sovereign AI Platform v18.8.0)
### الاسم الرمزي: "المُعِزّ" (Al-Mu'izz) - Universal Predator Edition

هذه هي المنصة الأكثر تقدماً للعمليات السيبرانية السيادية، تم دمجها الآن مع كافة أدوات **Kali Linux** و **BlackArch Linux** (أكثر من 3000 أداة) لتحويل جهازك إلى منظومة "المُعِزّ - المفترس" الكونية.

## 🚀 ما الجديد في نسخة المفترس الموحد (v18.8)؟
- **BlackArch Lexicon Sync:** دمج قاعدة بيانات تشمل 2800+ أداة هجومية متخصصة.
- **Universal Alpha Node:** عقل مركزي يفهم تصنيفات الهجوم المعقدة (NFC, Radio, VoIP, Firmware).
- **AI Model Harvester:** محرك بحث واستحواذ على موديلات الذكاء الاصطناعي المحلية وتوظيفها فوراً.
- **Predator UI v3:** واجهة هجومية غاشمة باللون الأحمر القتالي مع تأثيرات النبض العصبي المتطور.

## 🚢 نظام "سفينة نوح" (Noah's Ark v3)
وحدة النسخ الاحتياطي والتعافي الشامل لضمان الصمود السيادي.
- **الموقع:** `scripts/sovereign_ark_v3.sh`
- **التشغيل:** من الواجهة (System > Noah's Ark) أو عبر التيرمينال: `sov-backup`.

## متطلبات التشغيل الأساسية
- **نظام التشغيل:** Kali Linux 2024+ أو BlackArch Linux (منصح به بشدة).
- **المعالجة:** يفضل وجود GPU لعمليات الاستدلال المحلي، مع 8GB RAM كحد أدنى.
- **المتغيرات البيئية:** يجب وضع `GEMINI_API_KEY` في ملف `.env` لتفعيل العقل المركزي.

## بروتوكول التثبيت السريع
1. **منح صلاحيات التنفيذ:**
   ```bash
   chmod +x install.sh
   chmod +x scripts/*.sh
   ```
2. **بدء عملية التكوين:**
   ```bash
   sudo ./install.sh
   ```

## كيفية التشغيل
- **واجهة العمليات (Web Dashboard):** `npm run dev` (توجه إلى: `http://localhost:9002`)
- **مركز القيادة (TUI):** `sovereign`
- **الاستحواذ على النماذج:** من صفحة "System Pulse" اضغط على "Harvest Local Assets".
