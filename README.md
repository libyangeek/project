# منصة السيادة للذكاء الاصطناعي (Sovereign AI Platform v17.2.0)
### الاسم الرمزي: "المُعِزّ" (Al-Mu'izz) - Ultimate Edition

هذه هي المنصة الأكثر تقدماً للعمليات السيبرانية السيادية، مصممة لتحويل جهازك إلى منظومة "المُعِزّ" المتكاملة التي تدمج الذكاء الاصطناعي مع العمليات الهجومية والدفاعية.

## 🚢 نظام "سفينة نوح" (Noah's Ark v3)
يحتوي النظام على وحدة **Noah's Ark** للنسخ الاحتياطي والتعافي الشامل.
- **الموقع:** `scripts/sovereign_ark_v3.sh`
- **الوظيفة:** حفظ حالة النظام، الأكواد المولدة، قواعد بيانات الأدلة، ونماذج Ollama في أرشيف مشفر.
- **التشغيل:** من الواجهة (System > Noah's Ark) أو عبر التيرمينال: `sov-backup`.

## متطلبات التشغيل الأساسية
- **نظام التشغيل:** Kali Linux 2024+ (منصح به بشدة) أو Ubuntu 22.04+.
- **المعالجة:** يفضل وجود GPU لعمليات الاستدلال المحلي (Ollama).
- **البرمجيات:** Python 3.10+, NodeJS 18+, Docker.

## بروتوكول التثبيت السريع
1. **تحميل الملفات:** (انقل المجلد إلى جهازك).
2. **منح صلاحيات التنفيذ:**
   ```bash
   chmod +x install.sh
   chmod +x scripts/*.sh
   ```
3. **بدء عملية التكوين:**
   ```bash
   sudo ./install.sh
   ```

## كيفية التشغيل بعد التثبيت
- **واجهة العمليات (Web Dashboard):** `npm run dev` (توجه إلى: `http://localhost:9002`)
- **مركز القيادة (TUI):** `sovereign`
- **النسخ الاحتياطي:** `sov-backup`
