# منصة السيادة للذكاء الاصطناعي (Sovereign AI Platform v17.2.0)
### الاسم الرمزي: "المُعِزّ" (Al-Mu'izz) - Ultimate Edition

هذه هي المنصة الأكثر تقدماً للعمليات السيبرانية السيادية، مصممة لتحويل جهازك إلى منظومة "المُعِزّ" المتكاملة التي تدمج الذكاء الاصطناعي مع العمليات الهجومية والدفاعية.

## متطلبات التشغيل الأساسية

- **نظام التشغيل:** Kali Linux 2024+ (منصح به بشدة) أو Ubuntu 22.04+.
- **المعالجة:** يفضل وجود GPU لعمليات الاستدلال المحلي (Ollama).
- **البرمجيات:** Python 3.10+, NodeJS 18+, Docker.

## بروتوكول التثبيت السريع

لتحميل وتشغيل النظام على جهازك، اتبع الخطوات التالية في الطرفية (Terminal):

1. **تحميل الملفات:** (افترض أنك قمت بفك ضغط المجلد الخاص بالمنصة).
2. **منح صلاحيات التنفيذ:**
   ```bash
   chmod +x install.sh
   chmod +x scripts/*.sh
   ```
3. **بدء عملية التكوين (Sovereign Genesis):**
   ```bash
   sudo ./install.sh
   ```

## كيفية التشغيل بعد التثبيت

بمجرد اكتمال التثبيت، يمكنك تشغيل المكونات كالتالي:

- **واجهة العمليات (Web Dashboard):**
  ```bash
  npm run dev
  ```
  ثم توجه إلى: `http://localhost:9002`

- **مركز القيادة (TUI):**
  ```bash
  sovereign center
  ```

- **خادم الذكاء الاصطناعي (Inference Server):**
  ```bash
  python3 ai-engine/inference/server.py
  ```
