### 🛠 تعليمات التثبيت (الجهوزية الكاملة)

#### 🐧 لنظام Kali Linux / Linux:
1. افتح التيرمينال بصلاحيات Root.
2. نفذ الأمر: `sudo bash install.sh`
3. للإقلاع اليدوي (اختياري): `./muizz_start.sh`

#### 🪟 لنظام Windows 11:
1. افتح CMD أو PowerShell بصلاحيات Admin.
2. نفذ الأمر: `install.bat`
3. للإقلاع اليدوي (اختياري): `muizz_start.bat`
