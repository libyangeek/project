# منصة السيادة للذكاء الاصطناعي (Sovereign AI Platform v14.1.0)
### الاسم الرمزي: "المُعِزّ" (Al-Mu'izz)

هذه هي المنصة الأكثر تقدماً للعمليات السيبرانية الهجومية والدفاعية، مبنية على هندسة سيادية تضمن السيطرة الكاملة على البيانات والذكاء الاصطناعي.

## المكونات الأساسية

1. **محرك التوجيه الذكي (Smart Router):** يصنف طلبات المستخدمين ويوجهها لأفضل نموذج (Mistral, Qwen, etc.).
2. **وحدة OSINT Master:** لجمع المعلومات الاستخباراتية من المصادر المفتوحة (هاتف، بريد، نطاق).
3. **محرك Deep Eye:** فاحص ثغرات ويب متطور يكتشف SQLi, XSS برمجياً.
4. **خادم الاستدلال (Inference Server):** واجهة برمجة تطبيقات متوافقة مع OpenAI للاستخدام الداخلي الآمن.
5. **مركز القيادة (Command Center):** واجهة طرفية (TUI) للتحكم السريع في جميع الوحدات.

## كيفية التشغيل

المنصة مبنية باستخدام NextJS و Genkit و Python.

```bash
# تشغيل بيئة NextJS (الواجهة الرئيسية)
npm run dev

# تشغيل مركز القيادة التفاعلي
chmod +x scripts/command_center.sh
./scripts/command_center.sh
```

## المتطلبات السيادية
- `MISTRAL_API_KEY`: للوصول إلى محركات الاستدلال.
- `FIREBASE_CONFIG`: لإدارة قاعدة البيانات السيادية.
- `ADB`: لعمليات الفحص الجنائي للهواتف المحمولة.
