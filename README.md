# منصة السيادة للذكاء الاصطناعي (Sovereign AI Platform v14.1.0)
### الاسم الرمزي: "المُعِزّ" (Al-Mu'izz)

هذه هي المنصة الأكثر تقدماً للعمليات السيبرانية الهجومية والدفاعية، مبنية على هندسة سيادية تضمن السيطرة الكاملة على البيانات والذكاء الاصطناعي.

## المكونات الأساسية

1. **محرك التوجيه الذكي (Smart Router):** يصنف طلبات المستخدمين ويوجهها لأفضل نموذج (DeepSeek, Qwen, etc.).
2. **وحدة OSINT Master:** لجمع المعلومات الاستخباراتية من المصادر المفتوحة.
3. **محرك Deep Eye:** فاحص ثغرات ويب متطور يكتشف SQLi, XSS برمجياً.
4. **وحدة الهندسة الاجتماعية الآلية:** بوتات ذكية للتفاعل الإقناعي.
5. **مستودع المعرفة (RAG):** نظام استرجاع المعلومات من ملفات الأدلة الجنائية وسجلات الشات.

## كيفية التشغيل

المنصة مبنية باستخدام NextJS و Genkit.

```bash
# تثبيت التبعيات
npm install

# تشغيل بيئة التطوير
npm run dev

# تشغيل واجهة Genkit (لإدارة التدفقات)
npm run genkit:dev
```

## المتطلبات
- `GEMINI_API_KEY`: للوصول إلى محركات الاستدلال.
- `FIREBASE_CONFIG`: لإدارة قاعدة البيانات السيادية.
