# منصة السيادة للذكاء الاصطناعي (Sovereign AI Platform v14.1.0)
### الاسم الرمزي: "المُعِزّ" (Al-Mu'izz)

هذه هي المنصة الأكثر تقدماً للعمليات السيبرانية السيادية، مصممة للعمل في بيئات مغلقة ومؤمنة بالكامل.

## المكونات المدمجة

1.  **الموجه الذكي (Smart Router):** يحلل استفساراتك ويوجهها لأفضل نموذج (Mistral Large/Medium/Coder) لضمان دقة الاستجابة.
2.  **محرك Deep Eye:** فاحص ثغرات ويب متطور يكتشف (XSS, SQLi, LFI) باستخدام حمولات سيادية.
3.  **وحدة OSINT Master:** لجمع المعلومات من المصادر المفتوحة حول الهواتف، البريد، والنطاقات.
4.  **خادم الاستدلال (Inference Server):** واجهة API متوافقة مع OpenAI للاستخدام الداخلي عبر بروتوكول مؤمن.
5.  **مركز القيادة (Command Center):** واجهة طرفية (TUI) للتحكم السريع والفعال.

## متطلبات التشغيل

- **Python 3.9+** مع مكتبات: `flask`, `fastapi`, `requests`, `uvicorn`, `pydantic`.
- **NodeJS 18+** لتشغيل واجهة NextJS.
- **Firebase Project:** لإدارة البيانات والمصادقة السيادية.
- **ADB Tools:** لعمليات الفورنزكس الخاصة بالهواتف.

## كيفية البدء

```bash
# تثبيت المتطلبات
npm install
pip install fastapi uvicorn flask requests

# تشغيل مركز القيادة (الواجهة الرئيسية)
chmod +x scripts/command_center.sh
./scripts/command_center.sh
```

- `MISTRAL_API_KEY`: للوصول للمحرك اللغوي.
- `SOVEREIGN_SECRET`: مفتاح التشفير الخاص بالمنصة.
