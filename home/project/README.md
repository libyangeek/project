### 🛠 تعليمات التشغيل (إصدار التفرد المطلق)
بمجرد الإقلاع عبر `install.sh` الذكي، يفتح النظام "العرش الحي" تلقائياً داخل مصفوفة Docker المنيعة:
- **واجهة العرش (Golden HUD):** `http://localhost:9002` (وضع السيادة المطلقة)
- **الممر السيادي (API Bridge):** `http://localhost:8000` (النبض التنفيذي v63)

**أوامر السيطرة السريعة (Sovereign CLI):**
- `sovereign update`: تنفيذ ميثاق التثبيت v63.0 والارتقاء المادي الشامل.
- `sovereign strike <target>`: إطلاق هجوم الأسطول العليم المنسق عبر النبض النانوي.
- `sovereign nexus <target>`: تفعيل مصفوفة الافتراس الشامل (Predator Nexus).
- `sovereign recode <module>`: استدعاء الوكيل الميداني لإعادة برمجة النظام ذاتياً.
